# DA2411 Linear Algebra — Batch 24 Exam Prep 🎯

Hey everyone! This repo holds custom question banks built to help us get through the **DA2411 Final Paper**.

Instead of dumping random practice sheets, these markdown files are reverse-engineered from 6 years of past papers (2019–2025) to target the exact patterns, twists, and common traps our examiner actually uses.

## 🚨 IMPORTANT: HOW TO VIEW THE MATRICES 🚨

GitHub’s web interface can distort matrix formatting, making some solutions difficult to read.

**For the best experience:**

1. Clone or download this repository.
2. Open the folder in **VS Code**.
3. Press `Ctrl + Shift + V` (`Cmd + Shift + V` on Mac) to open the **Markdown Preview**.

This will render the matrices and formatting correctly.
